## Motivation 
    Freshman Utilities is an app created and developed for freshman who come into varsity life with a lot of things in their heart. 
    In order to answer their queries and help to build a good communication we created this app.
    With tools and features we created this app simply so that students  can understand these features and use them with comfort.
